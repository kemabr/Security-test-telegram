# JMeter Telegram Load Test Bot

Railway'de çalışan, gerçek JMeter entegreli Telegram botu.

## Kurulum

1. Railway'de yeni proje oluştur
2. GitHub repo'sunu bağla
3. Environment variables ekle:
   - BOT_TOKEN
   - WEBHOOK_URL
4. Deploy
